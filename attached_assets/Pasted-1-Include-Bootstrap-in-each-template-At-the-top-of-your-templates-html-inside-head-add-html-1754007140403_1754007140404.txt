1. Include Bootstrap in each template
At the top of your templates/*.html, inside <head>, add:

html
Copy
Edit
<link
  href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
  rel="stylesheet"
  integrity="sha384-…"
  crossorigin="anonymous"
/>
That gives you access to containers, cards, buttons, forms, etc.

2. Revised index.html
html
Copy
Edit
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Gradvantage – Grad Job Insights</title>
  <link
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
    rel="stylesheet"
  />
</head>
<body class="bg-light">
  <nav class="navbar navbar-dark bg-primary mb-4">
    <div class="container">
      <a class="navbar-brand" href="/">Gradvantage</a>
      <a class="btn btn-outline-light" href="/submit">Share Your Experience</a>
    </div>
  </nav>

  <div class="container">
    {% if submissions %}
      <div class="row gy-4">
        {% for e in submissions %}
          <div class="col-md-6">
            <div class="card shadow-sm">
              <div class="card-body">
                <h5 class="card-title">
                  {{ e.role }} at
                  <a href="/company/{{ e.company|urlencode }}">{{ e.company }}</a>
                </h5>
                <p class="card-text">
                  <strong>Outcome:</strong> {{ e.outcome }}<br>
                  <strong>Salary:</strong> {{ e.salary or '–' }},
                  <strong>Bonus:</strong> {{ e.bonus or '–' }}<br>
                  <strong>Uni:</strong> {{ e.university or '–' }}
                  (WAM: {{ e.wam or '–' }})
                </p>
                <p class="card-text"><em>{{ e.advice }}</em></p>
                <footer class="text-muted"><small>
                  Submitted: {{ e.timestamp.split('T')[0] }}
                </small></footer>
              </div>
            </div>
          </div>
        {% endfor %}
      </div>
    {% else %}
      <div class="alert alert-secondary text-center">
        No submissions yet. Be the first to share!
      </div>
    {% endif %}
  </div>
</body>
</html>
