I’m using Flask + Jinja. The “Application process” text on each experience card must be rendered as a visual, numbered timeline (no JS). Do these exact edits:

1) Replace the paragraph with inline Jinja timeline (no macros)

Open the template that renders each experience card (search in templates/ for the text Application process:). In that file, delete the paragraph that prints the raw process string, e.g.

Application process: {{ exp.application_process }}


(or experience.application_process)

Replace it with this block using whatever variable exists in this template (either exp or experience). If it’s experience, swap the variable names accordingly.

{# --- Application process timeline --- #}
{% set _process = (exp.application_process if exp is defined else experience.application_process) or '' %}
{% set _p = _process | replace('→','|') | replace('->','|') %}
{% set _steps = _p.split('|') | select('string') | list %}

{% if _steps|length and _steps|join('')|trim %}
<div class="process-card">
  <div class="process-head">
    <span class="pill pill-process">Application process</span>
  </div>

  <ol class="timeline">
    {% for s in _steps if s and s.strip() %}
    <li class="timeline-step">
      <span class="dot">{{ loop.index }}</span>
      <div class="step-card">{{ s.strip() }}</div>
    </li>
    {% endfor %}
  </ol>

  {% set _ix = (exp.interview_experience if exp is defined else experience.interview_experience) %}
  {% set _adv = (exp.advice if exp is defined else experience.advice) %}
  {% if _ix or _adv %}
  <details class="process-more">
    <summary>What to expect</summary>
    <ul class="ticks">
      {% if _ix %}<li>{{ _ix }}</li>{% endif %}
      {% if _adv %}<li>{{ _adv }}</li>{% endif %}
    </ul>
  </details>
  {% endif %}
</div>
{% endif %}
{# --- /Application process timeline --- #}
